# ETechnovetion

Follow these steps to run the project on Windows.

---

## Prerequisites
Install the following software:

- [**Git**](https://git-scm.com/downloads)
- [**Node.js** (version 24 or higher)](https://nodejs.org)

---

## Steps to Run the Project

1. **Fork the Project**  
   Open the project on GitHub and click **Fork** to copy it to your account.

2. **Clone Your Fork**  
   Open **Command Prompt** in the folder where you want the project and run:

   ```bash
   git clone https://github.com/<your-username>/ETechnovetion.git
   ```

3. **Change Directory**
   Move into the project folder:

   ```bash
   cd ETechnovetion
   ```

4. **Install Dependencies**
   Install the required packages:

   ```bash
   npm install
   ```
---

## How to Run the Development Server

Start the development server with:

```bash
npm run dev
```

The app will be available at [http://localhost:3000](http://localhost:3000).
